Создайте таблицу парковки получая данные с сервера (на сервере создайте массив для хранения данных).
 
Необходимые поля: номер машины, марка машины, ФИО хозяина, телефон, дата заезда, дата выезда.

Реализовать добавление машины - POST запрос на сервер (добавляем машину в созданный массив - память сервера).
 
Удаления отмеченных машин - можно выделить несколько строк (чекбоксы) и нажав 1 кнопку удалить сразу все выделенные.

Реализовать фильтрацию по названию машины - аналог контекстного поиска - ввели в строку символ/символы пошел запрос 
на сервер, результат обновление таблицы (оставлять только строки с вхождением строки поиска в название машины).
 
Если строка пустая - отображать все строки.
